💡 블렌더 실시간 소켓 연동 가이드

실시간 조립을 위해 블렌더 소켓 서버를 실행해야 합니다.

또는 수동 실행 방법:
  1. 우측 상단 복사(content_copy) 버튼을 눌러 스크립트를 복사합니다.
  2. 블렌더를 켜고, 상단 탭 맨 끝의 Scripting 메뉴를 클릭합니다.
  3. New 버튼을 클릭하여 새 문서를 만들고 코드를 붙여넣습니다.
  4. 상단의 Run Script (▶) 단추를 누르면 3D 형상이 생성됩니다!

API 키 설정 (AI 엔진 전용)

AI 모드를 사용하려면 Gemini API Key가 필요합니다. 입력된 키는 로컬 스토리지에만 보관됩니다.

무료 Gemini API Key 발급받기 →

API 키 권한에 맞거나 작동 가능한 모델을 직접 선택할 수 있습니다.

특정 환경용 커스텀 모델(예: gemini-3.5-flash-high)의 고유 API ID를 입력하세요.

Blender 연동 설정 (블렌더 모드 전용)

로컬에 설치된 Blender.exe 실행 경로 및 실시간 소켓 연동 방식을 구성합니다.

소켓 연동이 꺼져 있거나 연결에 실패할 때 백그라운드 빌드를 수행할 실행 경로입니다.

설정된 경로의 블렌더 프로그램을 실행하고 백그라운드 소켓 서버를 즉시 자동 구동합니다.

이 파일을 다운받아 블렌더 내 Scripting 탭에서 실행(▶)해 두시면 0.1초 만에 실시간으로 물체가 조립됩니다.
